Os conteudos a seguir foram montados com base nas aulas de formação de HTML.

Tags

Voltar

Tags são comandos utilizados para se montar uma estrutura HTML, existem diversas delas, cada uma com sua funcionalidade.

São utilizados o sinal de maior e menor para se construir uma tag.

Cada uma é correspondente algum elemento dentro da página.

O usuario só verá os contéudos presentes nas tags que estiverem dentro do Body. Ela é uma das principais tags, pois é responsavel por forma o esqueleto do documento.


Links

Voltar

Os links são fundamentais para permitir uma navegação entre páginas diversas, ou numa mesma página.

Facilitam uma boa navegação e usabiliade para os usuarios.

E ela é quem faz ancoras entre contéudos, é utilizado o atributo href, para permitir a linkagem e direcionamento.



Listas

Voltar

Existem dois tipos de listas presente no HTML, listas ordenadas (OL), e listas NÃO ordenadas (UL).

As ordenadas seguem uma ordem. Exemplo:

  1. Item
  2. Item
  3. Item

As não ordendas pelo contrario não segue uma ordem especifica, então são listas por um ponto ..

  • Item
  • Item
  • Item

No primeiro instante a lista pode não ter tanta utilidade mas a partir do CSS, podemos construir e manipula-las para ter funcionalidades incriveis.


Input

Voltar

O input cria diferentes campos de entrada de dados no documento HTML.

Existem outros inputs a serem abordados, mas vale lembra que cada tem sua aplicação.
